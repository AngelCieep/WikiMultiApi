POST Crear Nuevo Universo

Crea un nuevo universo en la base de datos.

Endpoint
POST /api/universes
URL de Prueba para Postman
Método: POST
URL: http://localhost:3000/api/universes
Body (JSON): Ver ejemplo completo abajo
Request Body (Campos Requeridos)
{
  "name": "Pokemon",
  "slug": "pokemon",
  "description": "Un mundo lleno de criaturas...",
  "logo": "https://ejemplo.com/logo.png",
  "backgroundImage": "https://ejemplo.com/bg.jpg",
  "primaryColor": "#FF0000",
  "secondaryColor": "#FFCC00",
  "popularityScore": 100,
  "releaseDate": "1996-02-27",
  "isActive": true
}
Campos Opcionales
  • imagenBoton (String): URL de imagen para botón
  • fontFamily (String): Fuente tipográfica
  • tertiaryColor (String): Color terciario
  • textColor (String): Color de texto
  • hasType (Boolean): Si tiene campo tipo
  • hasAbilities (Boolean): Si tiene campo habilidades
  • hasStats (Boolean): Si tiene campo estadísticas
  • statLabels (Object): Etiquetas personalizadas para estadísticas
{
  "name": "Pokemon",
  "slug": "pokemon",
  "description": "Un mundo lleno de criaturas mágicas llamadas Pokémon",
  "logo": "https://ejemplo.com/pokemon-logo.png",
  "backgroundImage": "https://ejemplo.com/pokemon-bg.jpg",
  "imagenBoton": "https://ejemplo.com/pokemon-btn.png",
  "fontFamily": "Pokemon Solid",
  "primaryColor": "#FF0000",
  "secondaryColor": "#FFCC00",
  "tertiaryColor": "#3B4CCA",
  "textColor": "#000000",
  "popularityScore": 100,
  "releaseDate": "1996-02-27",
  "isActive": true,
  "hasType": true,
  "hasAbilities": true,
  "hasStats": true,

    "ATK": "Ataque",

    "SPD": "Velocidad"

}
